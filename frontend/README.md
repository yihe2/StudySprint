# StudySprint Frontend

The frontend is a React + Vite single-page app for managing study goals and reviewing progress metrics.

## Development
- Install dependencies: `npm install`
- Start the dev server: `npm run dev`
- Build for production: `npm run build`
- Preview the production build: `npm run preview`

## Notes
- The Vite dev server proxies `/api` requests to the backend on `http://localhost:3001`
- The UI includes goal CRUD flows, bulk actions, filters, import/export, and Overview dashboard metrics
